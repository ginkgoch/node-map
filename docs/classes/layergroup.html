Class LayerGroup

This class represents a collection of a layers. It is used for organize the layer better in the app implementation.

For instance, we usually have a set of layers that are stable and not change very often; while another set of layers are stored in memory and change very often, just like dynamic editing or highlighting features. Then we could maintain the two sets of layers in two LayerGroup; then it is easier to manage their rendering or caching.
